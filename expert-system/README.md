# Проект экспертной системы оценки защищённости сегмента сети

**Тема:** Разработка проекта экспертной системы оценки уровня защищенности сегмента сети с моделями изолированных сессий рабочих станций

**Методология:** PMBOOK 6-е издание

**Связь с основным проектом:** Система изолированных сессий рабочих станций — прототип модели Zero Trust для образовательных организаций (ООО «Академия ТОП»).

---

## Описание

Данный проект содержит **проектную документацию** экспертной системы автоматической оценки уровня защищённости сегмента сети, использующего модель изолированных сессий рабочих станций. Система анализирует конфигурации, применяет базу знаний с правилами безопасности и вычисляет числовой **Индекс защищённости (0–100)** с вердиктом.

Проект разработан по методологии **PMBOK 6** и включает все необходимые артефакты: устав, ИСР, модель угроз, базу знаний, архитектуру, план тестирования.
